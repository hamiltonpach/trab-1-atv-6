# trab-1-atv-6
Esse código define uma estrutura chamada Aluno, que armazena nome, matrícula e média final. Em seguida, o programa cria três vetores: um para todos os alunos, um para os aprovados e outro para os reprovados.
Primeiramente, ele realiza a leitura dos dados de 10 alunos, utilizando um laço. O nome é lido com espaço, a matrícula como número inteiro e a média como número decimal.
Depois, o programa percorre o vetor de alunos e separa cada um conforme a média: alunos com média maior ou igual a 5 são colocados no vetor de aprovados, enquanto os demais vão para o vetor de reprovados. Essa separação é feita utilizando contadores para controlar a posição em cada vetor.
